Gravitex Paints 🎨

**Gravitex Paints** is a premium industrial and residential coating solution provider. This website serves as the digital storefront and portfolio for Gravitex, a subsidiary of **St. Marcel Homes & Engineering Ltd**. 

The platform is designed to showcase high-durability paint products, a project gallery, and a seamless quote request system for clients.

---

## 🚀 Features

* **Responsive Modern UI:** Fully mobile-optimized design with a custom-built "drawer" navigation menu.
* **Dynamic Gallery:** Interactive project showcase with category filtering (Interior, Exterior, Industrial).
* **Performance Counters:** Real-time animation of company milestones (Years Active, Projects Completed, etc.).
* **Lead Generation:** Integrated quote request forms with validation and submission states.
* **Smooth UX:** * Intersection Observer API for scroll-reveal animations.
    * Sticky navigation with backdrop blurring.
    * Lightbox integration for high-resolution project viewing.

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3 (Custom Variables/Grid/Flexbox), JavaScript (ES6+).
* **Animations:** CSS Transitions & Keyframes, Intersection Observer API.
* **Icons:** FontAwesome / Custom SVG icons.
* **Parent Organization:** St. Marcel Homes & Engineering Ltd.
